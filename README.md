# screenshot

You can take screenshot and recordings with it.
It's integrated with rofi.
What more do you want.


## Install

```bash
git clone https://github.com/runarsf/screenshot
cd screenshot
sudo make install
```

### Dependencies

- rofi
- ffcast
- xclip
- slop
- *sharenix

## Uninstall

```bash
sudo make uninstall
```
