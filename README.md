# screenshot

You can take screenshots and recordings with it. In different formats.
It's integrated with rofi.
What more do you want.


## Install

```bash
git clone https://github.com/runarsf/screenshot
cd screenshot
sudo make install
```

### Dependencies

- rofi
- ffcast
- xclip
- slop
- *sharenix

## Uninstall

```bash
sudo make uninstall
```
